# Todo App with Local Storage.
This is a practice prooject.I completed this simple app in two days.

To start:

```
clone this repo
git clone https://github.com/nsourov/Todo-LocalStorage.git
cd Todo-LocalStorage
npm install
npm start
go to http://localhost:3000
```
